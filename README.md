# nodejs-linux
installare nodejs su linux

# scaricare i files

```bash
git clone https://github.com/ArduinoDenis/nodejs-linux.git
```

# se usate apt-get 
```bash
cd nodejs-linux/ && sudo chmod 700 installer.sh && ./installer.sh
```

# se usate snap
```bash
cd nodejs-linux/ && sudo chmod 700 snap.sh && ./snap.sh
```
